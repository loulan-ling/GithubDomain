# 承影(chengying)

获取目标在GitHub上的子域和邮箱信息（获取目标在GitHub上的子域和邮箱信息）

这是斥候系统分解出来的一个功能，所以看起来有些许简陋。目前建议搜索域名时只是用主域名，不需要添加http等信息，比如想搜索google.com。直接输入google.com，而不是http://google.com。
